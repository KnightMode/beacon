What it does

One bot. The whole loop from question to pull request.

  1. Grounded Q&A/ask-code

    Streamed answers with sources. Abstains when evidence is missing.

  2. Agentic retrievalplanner

    Follow-up search, file reads, and graph walks when the first pass falls short.

  3. Hybrid searchBM25 + vectors + graph

    Full-text, semantic, and call-graph results — merged and reranked.

  4. PR reviewreview <url>

    Paste a PR URL. Get a review informed by the whole index.

  5. PR creation:rocket:

    Describe the issue, react 🚀 — Beacon opens the PR.

  6. CI-failure triageauto → Slack

    Failed runs get a cited diagnosis in your channel. React 🚀 for a fix PR.

  7. Automatic indexingGitHub App

    Install once. Every push reindexes only what changed.
